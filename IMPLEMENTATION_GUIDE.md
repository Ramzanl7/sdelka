- ✅ После подтверждения роль меняется на `shop`
- ✅ При отклонении роль меняется на `client`

## 🚀 Как использовать

### Для пользователей

#### Регистрация магазина
1. Перейдите на страницу регистрации
2. Выберите роль **"Магазин"**
3. Заполните форму регистрации
4. После регистрации ваш аккаунт получит статус **"ожидает подтверждения"**
5. Дождитесь подтверждения администратора

#### Оставление отзыва
1. Убедитесь, что заявка имеет статус **"Completed"** (Завершена)
2. Убедитесь, что вы участвовали в сделке (создатель, помощник, мастер, магазин)
3. Перейдите на страницу "Мои заявки"
4. Найдите завершенную заявку
5. Заполните форму оценки и оставьте отзыв

### Для администраторов

#### Вход в админ-панель
1. Войдите под учетной записью администратора:
   - **Email:** `admin@sdelka.local`
   - **Пароль:** `Admin123!`
2. Перейдите на страницу `admin.html` или в раздел "Админка" в меню

#### Подтверждение магазина
1. В админ-панели найдите секцию **"Заявки на подтверждение магазина"**
2. Просмотрите список ожидающих магазинов
3. Нажмите **"Подтвердить"** для одобрения или **"Отклонить"** для отказа
4. При подтверждении:
   - Роль пользователя меняется с `pending_verification` на `shop`
   - Пользователь получает полный доступ к функциям магазина
5. При отклонении:
   - Роль пользователя меняется на `client`
   - Пользователь становится обычным клиентом

#### Управление пользователями
1. В секции **"Пользователи"** отображаются все пользователи
2. Для каждого пользователя можно:
   - Изменить роль (кроме основного администратора)
   - Удалить пользователя (кроме основного администратора)
3. Роли пользователей:
   - `client` - Клиент
   - `master` - Мастер
   - `shop` - Магазин
   - `admin` - Администратор
   - `pending_verification` - Магазин, ожидающий подтверждения

### Проверки Anti-Fraud
```javascript
// 1. Заявка должна быть завершена
if (!request || request.status !== REQUEST_STATUS_COMPLETED) {
  return null;
}

// 2. Пользователь должен быть участником сделки
const isParticipant = 
  request.clientId === currentUser.id ||
  request.helperIds.includes(currentUser.id) ||
  request.masterId === currentUser.id ||
  request.shopId === currentUser.id ||
  request.completedBy === currentUser.id;

if (!isParticipant) {
  return null;
}

// 3. Один отзыв на сделку от пользователя
const hasExistingRating = existingRatings.some(r => 
  r.requestId === requestId && 
  r.fromUserId === currentUser.id
);

if (hasExistingRating) {
  return null;
}
```

### Сценарий 2: Подтверждение магазина администратором
1. Войдите как администратор (`admin@sdelka.local / Admin123!`)
2. Перейдите в админ-панель
3. Найдите секцию "Заявки на подтверждение магазина"
4. Нажмите "Подтвердить" для ожидающего магазина
5. **Ожидаемый результат:** Роль пользователя изменена на `shop`

### Сценарий 3: Попытка оставить отзыв для незавершенной сделки
1. Создайте заявку
2. Попытайтесь оставить отзыв
3. **Ожидаемый результат:** Отзыв не сохраняется (заявка не завершена)

### Сценарий 4: Попытка оставить отзыв не участником сделки
1. Создайте заявку от имени пользователя A
2. Войдите как пользователь B (не участвующий в сделке)
3. Завершите заявку
4. Попытайтесь оставить отзыв
5. **Ожидаемый результат:** Отзыв не сохраняется (пользователь не участник)

1. Создайте и завершите заявку
2. Войдите как участник сделки
3. Оставьте отзыв
4. Попытайтесь оставить второй отзыв на ту же заявку
5. **Ожидаемый результат:** Второй отзыв не сохраняется

## 🆘 Устранение неполадок

### Проблема: Магазины не отображаются в секции подтверждения
**Решение:** Убедитесь, что:
1. Пользователь зарегистрировался с ролью "Магазин"
2. Вошел в систему под этим пользователем хотя бы раз
3. Администратор вошел в админ-панель

### Проблема: Нельзя оставить отзыв
**Решение:** Проверьте:
1. Статус заявки должен быть `completed`
2. Вы должны быть участником сделки
3. Вы еще не оставляли отзыв на эту заявку

### Проблема: Разные демо-админы в разных частях приложения
**Решение:** Очистите localStorage и перезагрузите страницу:
```javascript
localStorage.clear();
location.reload();
```
